SIT (alias "Supportinfo-Tool")
Ein Freeware-Programm von Philipp Meisberger (PM Code Works)

"SIT ist ein Tool zum einfachen Eintragen von System-Herstellerinformationen in den 
Systemsteuerungszweig System. IT-Firmen können so mit wenigen Klicks ihren (virtuellen) 
Firmenstempel auf dem Kunden PC hinterlassen."
